Random Seed

This is the web implementation of the Random Seed Algorithm, used to solve complex linear programming problems. Of course, it's a iterative algorithm based on a number of populations, each one with the same size. So, with this tool you can enter the populations info, the restrictions and the objective function. It will display all the iterations and found the most accurate result.

Star Cluster

This is a unfinished project that implements the Brute Force Star Cluster Algorithm. From a big population of stars in the sky (actually any bunch of points in a 2D space), it finds the two of them with the minimum distance between each other. We put an animation of how the algorithm is supossed to be working when it calculates the distance from point to point; as you can imagine, is zero optimized and takes a lot of time... but looks pretty good!

Sniffer

A network packet analyzer Java application, better known as sniffer. With the jnetpcap library, the application catches frames, classifies it with it's corresponding protocol and show it's headers. The application show a graph with the statics of all the packets caught. Also, you can configure the network card of your device, set filters and import/export frames in .pcap files.
